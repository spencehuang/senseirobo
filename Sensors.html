Common Sensors in Robotics

    Ultrasonic Sensors:

    Ultrasonic sensors detect distance between itself and the object in front of it. Like a bat, it outputs a wave and calculates the time it takes for it to bounce back. These motors are commonly used in moving machines like automobiles, dispensers, screen buttons, and drink filling machines.

    --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

    Ring LEDs:

    These LEDs emit lights of different colors. They are commonly used for basic lighting and indicating amounts. The have a large and round light source which are usually used with detectors to show amount.

    Some projects using this would be material level detecters and battery level monitors.

    --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

    LED Screens:

    These LED screens have multiple single color LEDs which emit lights of different colors. They are commonly used communication and showing images based on the code. The have a large and round light source which are usually used with detectors to show amount.

    Some projects using this would be robots with facial expressions, mini arcade games, and signage.
